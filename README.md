# e-plantShopping

## Paradise Nursery Plant Shop

e-plantShopping is a React and Redux based online plant shopping application for Paradise Nursery.

### Features

- Landing page with Paradise Nursery company name
- Get Started button
- Product listing page
- Add to Cart functionality
- Shopping cart page
- Increase and decrease product quantity
- Remove items from cart
- Redux Toolkit cart state management

### Technologies Used

- React
- Redux Toolkit
- React Router
- CSS
